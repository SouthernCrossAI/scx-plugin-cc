# Contributing

Thanks for your interest in **samba-plugin**. Please read this before opening an issue or pull request.

## How this project is developed

samba-plugin is developed internally at SambaNova and published here as a **public mirror**. Development happens on our internal tree; this repository is the released, read-only-for-contributions copy.

### Issues — welcome 👍

We actively want your bug reports and feature requests. Please use the issue templates:

- **Bug report** — include the plugin version (`plugins/samba-plugin/.claude-plugin/plugin.json`), your OS, Claude Code version, exact steps to reproduce, and what you expected vs. what happened.
- **Feature request** — describe the problem you're trying to solve, not just a proposed solution.

A good issue is reproducible and specific. Issues that are vague or that we can't reproduce may be closed.

### Pull requests — not accepted ✋

Because development is internal, **we do not accept external pull requests.** Any PR opened against this repository will be automatically closed with a pointer back to this guide. This is not a judgment on your change — please file an **issue** describing it instead, and we'll triage it into our internal workflow.

## Security

**Do not report security vulnerabilities in public issues.** Contact the SambaNova team privately instead.
